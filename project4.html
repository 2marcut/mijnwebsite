React Dashboard

Technologieën: React, Node.js, API-integratie

Dit dashboard is een interactieve webapplicatie gebouwd in React. Het toont real-time data, grafieken en widgets die automatisch verversen via API-calls.

Het project leerde mij hoe component-gedreven ontwikkeling werkt en hoe state management een belangrijke rol speelt in moderne webapplicaties. Ook leerde ik JSON-data verwerken en dynamisch weergeven.
