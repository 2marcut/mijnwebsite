E-Commerce Website

Technologieën: HTML, CSS, JavaScript

Dit project is een volledige webshop die ik zelf heb ontworpen en gebouwd. De website bevat productpagina’s, een winkelmandje, dynamische scripts en een moderne layout. Het doel was een gebruiksvriendelijke en responsieve webshop te creëren.

Tijdens dit project leerde ik werken met gestructureerde HTML-componenten, geavanceerde CSS-layouts en JavaScript-functionaliteiten zoals het updaten van het winkelmandje in real time.
