Smart Home Dashboard

Technologieën: HTML, CSS, JavaScript, IoT API’s

Dit project is een dashboard waarmee gebruikers IoT-apparaten in hun huis kunnen beheren via een overzichtelijke webinterface. Je kunt lampen bedienen, temperatuur instellen en apparaten monitoren.

Ik bouwde interactieve componenten en gebruikte API’s voor real-time dataverwerking. De focus lag op overzichtelijkheid, veiligheid en eenvoudige bediening.
